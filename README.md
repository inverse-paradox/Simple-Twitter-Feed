Simple-Twitter-Feed
===================

WordPress Plugin: Simple Twitter Feed by Inverse Paradox; results are cached to avoid excessive queries.

# Simple useage
    [ip_twitter user="InverseParadox"]

# Other accepted params

    num (default = 5)
    before (default = "<li>")
    after (default = "</li>")
    widget_before (default = "<ul>")
    widget_after (default = "</ul>")
